# Schemaläggningsassistent - Frontend

React frontend for AI-powered healthcare scheduling.

## Setup
```bash
npm install
```

## Run locally
```bash
npm run dev
```

Runs on `http://localhost:5173`

Backend must be running on `http://localhost:5000`

## Build
```bash
npm run build
```

## Deploy
```bash
npm run deploy
```

Deployed to GitHub Pages: https://alterot.github.io/schema-frontend/

## Tech

React 18, Vite, vanilla CSS
